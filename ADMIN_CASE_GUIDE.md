# 📦 Как создать первый кейс

## 👋 Привет!

Теперь в игре **нет дефолтных кейсов**. Все кейсы создаются администраторами (developer, first_developer).

---

## 🎯 Инструкция по созданию кейса

### 1. Открой админ-панель

```
Главное меню → Админ (🛠️)
```

**Доступно только ролям:**
- developer 👨‍💻
- first_developer 👑 Mixazx

---

### 2. Перейди во вкладку "Кейс"

```
Админ → 📦 Кейс
```

---

### 3. Заполни информацию

**Поля:**
- **Название кейса:** например "Мифический кейс"
- **Цена:** например 10000
- **Валюта:** KIT или Crypto
- **Редкость:** выбери одну из:
  - Все редкости (ручной выбор скинов)
  - Common (Обычный)
  - Rare (Редкий)
  - Epic (Эпический)
  - Legendary (Легендарный)
  - Mythical (Мифический)

---

### 4. Выбери скины

**Если выбрал "Все редкости":**
- Кликни на каждый скин который хочешь добавить
- Можно загрузить своё изображение для каждого скина (кнопка "📷 Загрузить")

**Если выбрал конкретную редкость:**
- Все скины этой редкости автоматически добавятся в кейс

---

### 5. Создай кейс

Нажми кнопку **"Создать кейс"** внизу страницы.

**Готово!** Кейс появится во вкладке "KIT Кейсы" или "Crypto Кейсы".

---

## 🎨 Загрузка изображений для скинов

### Для каждого скина можно загрузить картинку:

1. Найди скин в списке
2. Нажми **"📷 Загрузить"**
3. Выбери файл (PNG, JPG, GIF, WEBP)
4. Максимум 2MB
5. Дождись загрузки

**Изображение сохранится в localStorage!**

---

## 📋 Примеры кейсов

### Кейс "Новичок" (KIT)
```
Название: Новичок
Цена: 100
Валюта: KIT
Редкость: Common
```

### Кейс "Элита" (Crypto)
```
Название: Элитный
Цена: 50
Валюта: Crypto
Редкость: Legendary + Mythical
```

### Кастомный кейс
```
Название: Мой кейс
Цена: 5000
Валюта: KIT
Редкость: Все редкости
Скины: [выбрать вручную 5-10 скинов]
```

---

## ✅ Проверка

После создания:

1. **Открой:** KIT Кейсы или Crypto Кейсы
2. **Найди:** свой кейс в списке
3. **Купи:** если хватает денег
4. **Открой:** проверь что скины выпадают

---

## 💡 Советы

- ✅ Начинай с дешёвых кейсов (100-500 KIT)
- ✅ Добавляй 3-5 скинов разной редкости
- ✅ Загружай красивые изображения
- ✅ Тестируй каждый кейс перед публикацией

---

## 🔧 Если что-то не работает

### Кейс не создаётся:
- Проверь что заполнены все поля
- Проверь что выбран хотя бы один скин
- Обнови страницу и попробуй снова

### Изображение не загружается:
- Проверь размер файла (макс 2MB)
- Проверь формат (PNG, JPG, GIF, WEBP)
- Попробуй другой файл

### Кейс не отображается:
- Проверь что выбран правильную валюту (KIT/Crypto)
- Обнови страницу
- Проверь что кейс создан успешно
