# 🖼️ Инструкция по загрузке картинок

## 📋 Как загрузить картинки

### Способ 1: Через приложение (Рекомендуется)

1. **Откройте админ-панель**
   - Запустите приложение
   - Нажмите на вкладку "Админ" или "Картинки" (доступно только разработчикам)

2. **Перейдите во вкладку "Картинки"**
   - Выберите тип: 🐱 Кот / 📦 Кейс / 🎨 Скины

3. **Загрузите изображение**
   - **Файл**: Нажмите "Выбрать файл" и выберите картинку на устройстве
   - **URL**: Вставьте ссылку на изображение и нажмите "Загрузить по URL"

4. **Сохраните**
   - Нажмите "💾 Сохранить"
   - Страница перезагрузится автоматически

### Способ 2: Через файловую систему

1. **Подготовьте изображения**
   - Формат: PNG, JPG, GIF, WEBP, SVG
   - Размер: до 5MB
   - Рекомендуемое разрешение: 512x512px

2. **Поместите файлы в папку `public/`**
   ```
   cat/
   └── public/
       ├── default_cat.svg    ← Дефолтный кот (уже есть)
       ├── my_cat.png         ← Ваш кот
       ├── case1.png          ← Кейс 1
       └── skin_legendary.png ← Скины
   ```

3. **Обновите `src/data/gameData.ts`**
   ```typescript
   // Для кота
   { id: 'cat_1', name: 'Дворовый', image: '/my_cat.png', ... }
   
   // Для кейса
   { id: 'case_starter', name: 'Новичок', image: '/case1.png', ... }
   
   // Для скина
   { id: 'skin_10', name: 'Золотой', image: '/skin_legendary.png', ... }
   ```

## 📁 Дефолтные файлы

### Кот (по умолчанию)
**Файл:** `public/default_cat.svg`

**Используется когда:**
- Не загружено кастомное изображение
- Первый запуск приложения

**Как использовать свой файл:**
1. Назовите файл: `custom_cat.png` (или любое другое название)
2. Положите в папку `public/`
3. В приложении загрузите через вкладку "Картинки"

---

## 🎨 Рекомендации к изображениям

### Для кота (главная кнопка)
- **Размер:** 512x512px
- **Формат:** PNG с прозрачным фоном или SVG
- **Стиль:** Круглое или квадратное изображение
- **Пример:** `cat.png`, `main_cat.webp`

### Для кейсов
- **Размер:** 256x256px
- **Формат:** PNG с прозрачным фоном
- **Стиль:** Изометрия или вид спереди
- **Пример:** `case_starter.png`, `mythic_case.webp`

### Для скинов
- **Размер:** 128x128px
- **Формат:** PNG с прозрачным фоном
- **Стиль:** Иконка предмета
- **Пример:** `skin_common.png`, `rare_cat.webp`

### Для котов в инвентаре
- **Размер:** 256x256px
- **Формат:** PNG с прозрачным фоном
- **Стиль:** Полное изображение кота
- **Пример:** `cat_business.png`, `cat_magnat.webp`

## 💾 Где хранятся загруженные картинки

### LocalStorage (браузер)
- **Ключ:** `custom_image_cat`, `custom_image_case`, `custom_image_skin`
- **Формат:** Base64 (Data URL)
- **Размер:** До 5MB на изображение
- **Где:** В браузере пользователя/устройства

### Файловая система (проект)
- **Папка:** `public/`
- **Доступ:** По URL `/filename.png`
- **Пример:** `<img src="/my_cat.png" />`

## 🔄 Сброс изображений

### Через приложение:
1. Откройте вкладку "Картинки"
2. Выберите тип изображения
3. Нажмите "🗑️ Сбросить"

### Вручную (LocalStorage):
```javascript
localStorage.removeItem('custom_image_cat');
localStorage.removeItem('custom_image_case');
localStorage.removeItem('custom_image_skin');
```

### Полная очистка:
```javascript
localStorage.clear();
```

## 🛠️ Для разработчиков

### Интеграция в код:

```typescript
// Загрузка из localStorage
const customImage = localStorage.getItem('custom_image_cat');

// Использование в компоненте
<img src={customImage || '/default_cat.svg'} alt="Cat" />
```

### Обновление gameData.ts:

```typescript
export const CATS: Cat[] = [
  {
    id: 'cat_1',
    name: 'Дворовый',
    rarity: 'common',
    image: '/cats/cat_default.png',  // ← Ваш путь
    passiveIncome: 10,
    description: 'Простой кот'
  }
];
```

## ⚠️ Важно

1. **Размер файлов** - не более 5MB для загрузки через приложение
2. **Форматы** - PNG, JPG, GIF, WEBP, SVG
3. **LocalStorage** - очищается при очистке кэша браузера
4. **Vercel** - при деплое используйте папку `public/` для постоянных файлов
5. **Мобильные** - на телефонах лучше использовать URL загрузку
