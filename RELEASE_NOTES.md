## 🎮 Как играть теперь

### Для обычных пользователей:
1. Открой игру
2. Тапай кота
3. Зарабатывай KIT
4. **Жди пока админ создаст кейсы**
5. Покупай и открывай кейсы

### Для админов (developer):
1. Открой админ-панель
2. Перейди во вкладку "Кейс"
3. Создай первый кейс
4. Загрузи изображения для скинов
5. **Игроки смогут покупать!**

## 🚀 Что дальше

### 1. Протестируй локально
```bash
npm run dev
```

### 2. Создай первый кейс (как админ)
- Открой админку
- Создай кейс "Новичок" (100 KIT, Common)
- Создай кейс "Элита" (50 Crypto, Epic+)

### 3. Проверь работу
- Купи кейс
- Открой кейс
- Проверь что скин выпал

### 4. Задеплой на Vercel
```bash
vercel
```
