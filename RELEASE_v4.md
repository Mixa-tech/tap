# 🚀 Cat Tapper - Telegram Web App - RELEASE v4.0

## 🎮 Полное описание

**Cat Tapper** - это многофункциональная Telegram Web App игра в стиле TONcoin с механикой кликера, кейсами, биржей, маркетплейсом и системой администрирования.

---

## ✨ Все возможности

### 🐱 Основное
- **Тапанье кота** - кликай и зарабатывай KIT
- **Перерождения** - до 100 уровней с бонусом +10% за каждое
- **50 ачивок** - достижения во всех категориях
- **Скины** - 15 скинов с 5 редкостями (Common → Mythical)

### 📦 Кейсы
- **Кейсы за KIT** - 4 стартовых кейса
- **Кейсы за Crypto** - 4 кейса в стиле CS2 с рулеткой
- **Создание кейсов** - developer может создавать свои кейсы
- **Настройка редкости** - выбор конкретной редкости или всех

### 💰 Криптовалюты
- **KIT** - основная валюта
- **Пользовательские валюты** - админы создают сами
- **Курсы валют** - отслеживание курсов и изменений
- **Управление ценами** - админы могут менять цены

### 💱 Биржа
- **Покупка/продажа** - торговля всеми валютами
- **Калькулятор** - расчёт стоимости сделки
- **Проверка баланса** - контроль средств

### 🏪 Маркетплейс
- **Покупка скинов** - покупка у других игроков
- **Продажа скинов** - выставление своих скинов
- **Выбор валюты** - KIT или Crypto
- **Комиссия 5%** - с каждой продажи (0% для премиум)
- **Разные валюты** - KIT или Crypto

### 🎒 Инвентарь
- **Коты** - пассивный доход KIT (10-250/час)
- **Скины** - коллекция с бонусами к клику
- **Видеокарты** - майнинг криптовалюты
- **Сборка фермы** - расширенный процесс сборки

### 🛠️ Админ-панель
- **Создание кейсов** - с выбором редкости и скинов
- **Создание валют** - добавление новых криптовалют
- **Выдача ролей** - user, premium, developer
- **Выдача денег** - KIT, Crypto или пользовательские валюты
- **Загрузка картинок** - для котов, кейсов, скинов
- **Управление ценами** - изменение курсов криптовалют

### ⭐ Premium функции
- **+20% опыта** - ускоренная прокачка
- **+20% денег** - увеличенный заработок
- **Комиссия 0%** - продажа скинов без комиссии
- **Скидки 10%** - на покупку криптовалют

## 💰 Экономика

### Валюты
- **KIT** - основная валюта (клики, пассивный доход)
- **Crypto** - премиум валюта (майнинг, кейсы)
- **Пользовательские** - создаются админами

### Источники дохода
- Клик по коту: 1-55 KIT (+20-50% для премиум)
- Коты: 10-250 KIT/час
- Майнинг: 0.000025-0.0005 Crypto/час
- Ачивки: 10-100,000 KIT
- Маркетплейс: продажа скинов

### Премиум бонусы
- **Premium**: +20% XP, +20% денег, 0% комиссия, 10% скидки
- **Developer**: +50% XP, +50% денег, 0% комиссия, 20% скидки

## 🖼️ Загрузка картинок

### Через приложение
1. Открыть вкладку "Картинки"
2. Выбрать тип (Кот / Кейс / Скины)
3. Загрузить файл или URL
4. Сохранить

### Через файловую систему
1. Поместить файл в `public/`
2. Обновить `src/data/gameData.ts`

**Дефолтный файл кота:** `public/default_cat.svg`

## 🔧 Для разработчиков

### Админ-панель
Доступна только ролям `developer` и `first_developer`:

#### 6 вкладок админки:
1. **📦 Кейс** - создание с выбором редкости
2. **💰 Создать** - создание криптовалюты
3. **👥 Роль** - выдача ролей
4. **💵 Деньги** - выдача денег
5. **🖼️ Картинки** - загрузка изображений
6. **📊 Управление** - изменение цен валют

#### Создание криптовалюты:
```typescript
createCurrency({
  name: 'Bitcoin',
  symbol: 'BTC',
  price: 65000,
  change24h: 2.5,
  icon: ''
});
```

### Добавление скинов
```typescript
// src/data/gameData.ts
export const SKINS: Skin[] = [
  {
    id: 'skin_new',
    name: 'Новый скин',
    rarity: 'legendary',
    chance: 5,
    image: '/skins/new_skin.png',
    bonus: 2.5,
    description: 'Описание скина'
  }
];
```
